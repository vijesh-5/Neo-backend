## Installation

Clone the repository:

```bash
git clone https://github.com/vijesh-5/Neo-backend.git
````

Install dependencies:

```bash
npm install
```

Create a `.env` file in the root and add:

```
PORT=5000
MONGO_URI=your_mongodb_connection
```

Run the server:

```bash
npm run dev
```

or

```bash
npm start
```
